Uasabi is a software consultancy delivering testable and predictable code from London, UK

In the past decade Uasabi has been practising software development with a focus on testing and automation. Our expertise has been recognised through our work with companies in the financial and public sector.

Uasabi believes in creating meaningful work through values

  • 1

    People-first engineering

    Writing software is just a means to an end. Users don't ask for software. They want to be able to achieve something, solve problems or make their life easier. We should design solutions without forgetting to put compassion, empathy and inclusion at the forefront.

  • 2

    Predictable code

    Code isn't art or poetry. It's purposeful. It's a solution to a problem. We should try to write code in a way that is as uncluttered, simple and non-egotistical as possible. We should aim to make our code easy to understand and adapt.

  • 3

    Pragmatism

    We should design solutions that are easy to change and embrace uncertainty. That's not bad or an inconvenience, it's our reality. When it comes to getting things done "First do it, then do it right, then do it better" is the mantra for successfully delivering software.

  • 4

    Candid communication

    Speaking frankly to each other, without unnecessary embellishments or polite veneers is meant to help people acknowledge and learn from each other. Showing tolerance, curiosity, the desire to learn and respect, is the first step to build a durable connection.
